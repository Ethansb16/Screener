2. In `src/db/schema.js`, inside `initializeSchema()`, add the following SQL block AFTER the `run_log` table creation and BEFORE the closing `);` of `db.exec()`:

```sql
    CREATE TABLE IF NOT EXISTS signals (
      id           INTEGER PRIMARY KEY AUTOINCREMENT,
      filing_id    INTEGER NOT NULL REFERENCES filings(id),
      signal_name  TEXT    NOT NULL,
      classification TEXT,
      confidence   TEXT,
      raw_excerpt  TEXT,
      extracted_at TEXT DEFAULT (datetime('now')),
      UNIQUE(filing_id, signal_name)
    );
    CREATE INDEX IF NOT EXISTS idx_signals_filing_id ON signals(filing_id);
    CREATE INDEX IF NOT EXISTS idx_signals_signal_name ON signals(signal_name);
```

The `signal_name` column values will be one of: `'reason_classification'`, `'equity_grants'`, `'debt_loading'`, `'management_continuity'`.

The `UNIQUE(filing_id, signal_name)` constraint enables INSERT OR REPLACE idempotency on re-runs.
